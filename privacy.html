Koto – Tietosuojaseloste

Päivitetty 22.4.2026

1. Rekisterinpitäjä

Koto-sovelluksen kehittäjä Olli Paananen
Sähköposti: ollip.paananen@gmail.com

2. Kerättävät tiedot

Koto kerää vain tiedot jotka käyttäjä itse syöttää:

  • Tilitiedot: sähköpostiosoite ja salasana (salattu)
  • Kiinteistön tiedot: osoite, rakennusvuosi, rakenteet, talotekniikka, kuva
  • Huoltotiedot: tehtävät, päiväkirjamerkinnät, dokumentit, kustannukset

Sovellus ei kerää sijaintitietoja, analytiikkaa, mainostunnuksia eikä jaa tietoja kolmansille osapuolille.

3. Tietojen tallennus

Tiedot tallennetaan Supabase-pilvipalveluun (EU, Frankfurt). Jokainen käyttäjä näkee vain omat tietonsa (Row Level Security). Salasanat on salattu eikä niitä tallenneta selkokielisinä.

4. Tietojen poistaminen

Käyttäjä voi poistaa kiinteistöjään, tehtäviään ja dokumenttejaan sovelluksessa. Tilin ja kaiken datan poiston voi pyytää sähköpostilla: ollip.paananen@gmail.com

5. Muutokset

Tätä selostetta voidaan päivittää. Merkittävistä muutoksista ilmoitetaan sovelluksessa.

Koto – Privacy Policy

Updated April 22, 2026

1. Data Controller

Koto app developer Olli Paananen
Email: ollip.paananen@gmail.com

2. Data Collected

Koto only collects data that the user enters:

  • Account: email address and password (encrypted)
  • Property details: address, year built, structures, building systems, photo
  • Maintenance data: tasks, diary entries, documents, costs

The app does not collect location data, analytics, advertising identifiers, nor does it share data with third parties.

3. Data Storage

Data is stored in Supabase cloud service (EU, Frankfurt). Each user can only access their own data (Row Level Security). Passwords are encrypted and never stored in plain text.

4. Data Deletion

Users can delete their properties, tasks and documents within the app. To request full account and data deletion, email: ollip.paananen@gmail.com

5. Changes

This policy may be updated. Significant changes will be communicated within the app.

Koto – Tietosuojaseloste

Päivitetty 26.4.2026

1. Rekisterinpitäjä

Koto-sovelluksen kehittäjä Olli Paananen
Sähköposti: ollip.paananen@gmail.com

2. Kerättävät tiedot

Käyttäjän itse syöttämät tiedot:

  • Tilitiedot: sähköpostiosoite ja salasana (salattu)
  • Kiinteistön tiedot: osoite, rakennusvuosi, rakenteet, talotekniikka, kuva
  • Huoltotiedot: tehtävät, päiväkirjamerkinnät, dokumentit, kustannukset

Lisäksi sovelluksen käytöstä kerätään anonymisoitua tilastotietoa palvelun kehittämiseksi:

  • Sovelluksen avaukset ja näkymien käyttö
  • Toimintojen käyttö (esim. tehtävän lisäys, dokumentin tallennus, raportin lataus)
  • Mahdolliset virhetilanteet

Tilastotiedot eivät sisällä kiinteistön tietoja, dokumenttien sisältöjä tai muuta käyttäjän syöttämää aineistoa. Käyttäjä yksilöidään anonyymillä tunnisteella.

Sovellus ei kerää sijaintitietoja eikä mainostunnuksia.

3. Tietojen tallennus ja jakaminen

Käyttäjän tiedot tallennetaan Supabase-pilvipalveluun (EU, Frankfurt). Jokainen käyttäjä näkee vain omat tietonsa (Row Level Security). Salasanat on salattu eikä niitä tallenneta selkokielisinä.

Käytön tilastotietoja käsittelee PostHog (EU). Tiedot eivät yhdisty kiinteistön sisältötietoihin.

Tietoja ei myydä eikä jaeta kolmansille osapuolille markkinointitarkoituksiin.

4. Tietojen poistaminen

Käyttäjä voi poistaa kiinteistöjään, tehtäviään ja dokumenttejaan sovelluksessa. Tilin ja kaiken datan voi poistaa pysyvästi sovelluksen Profiili-sivulta, tai pyytää sähköpostilla: ollip.paananen@gmail.com

5. Muutokset

Tätä selostetta voidaan päivittää. Merkittävistä muutoksista ilmoitetaan sovelluksessa.

Koto – Privacy Policy

Updated April 26, 2026

1. Data Controller

Koto app developer Olli Paananen
Email: ollip.paananen@gmail.com

2. Data Collected

Data the user enters:

  • Account: email address and password (encrypted)
  • Property details: address, year built, structures, building systems, photo
  • Maintenance data: tasks, diary entries, documents, costs

In addition, anonymous usage statistics are collected to improve the service:

  • App opens and screen views
  • Feature usage (e.g. task added, document saved, report exported)
  • Errors and crashes

Usage statistics do not include property details, document contents or any other user-entered data. Users are identified by an anonymous identifier.

The app does not collect location data or advertising identifiers.

3. Data Storage and Sharing

User content is stored in Supabase (EU, Frankfurt). Each user can only access their own data (Row Level Security). Passwords are encrypted and never stored in plain text.

Usage statistics are processed by PostHog (EU). They are not linked to property content.

Data is not sold or shared with third parties for marketing purposes.

4. Data Deletion

Users can delete their properties, tasks and documents within the app. The account and all data can be permanently deleted from the app's Profile page, or requested by email: ollip.paananen@gmail.com

5. Changes

This policy may be updated. Significant changes will be communicated within the app.
